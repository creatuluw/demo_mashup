About this Mashup

Why should you be interested?

When you are used to developing with Qlikview or used to developing stuff with webtechnology, this might be an interesting application/mashup for you. It is build on top of the data found with this app on the Qlik demosite. A redesign of this app as a result of using that data.

But most interesting is the Qlik Analytics Platform which is responsible for this result. This platform enables developers and tech enthusiasts to build pure web applications, combining data, visualizations and interactivity into one solution.

If you want to get started there are, next to this demo with source code, a few resources you could visit.
